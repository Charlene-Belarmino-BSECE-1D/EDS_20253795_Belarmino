# Silica Impurity Fluctuations in Iron Ore Flotation Concentrates

A statistical engineering framework designed to automate industrial mining telemetry ingestion, cleaning, and descriptive distribution analytics.

## Project Structure
* `main.py`: Core object-oriented automated processing pipeline.
* `requirements.txt`: Environment configuration for reproducible research.
* `data/`: Protected directory containing raw telemetry csv files.
* `outputs/`: Individual static and dynamic system simulation plots.

## System Prerequisites
To install the necessary processing engines, navigate to the project directory and run:

pip install -r requirements.txt
